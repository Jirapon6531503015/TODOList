# Backend Java

This is the backend for the my-todo-project. It uses Spring Boot and Maven.

## Getting Started

1. Install Java (LTS recommended)
2. Run `mvnw spring-boot:run` to start the backend server.

## Project Structure
- `src/main/java` - Source code
- `src/main/resources` - Configurations
- `src/test/java` - Test code
